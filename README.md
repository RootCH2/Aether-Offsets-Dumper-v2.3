# Aether-Offsets-Dumper-v2.3
## 🚀 Features -           
✔ Extracts module base + offsets
- ✔ Works with any Windows process (enter exact name)
-   ✔ Bulletproof scanning system
- ✔ Small, clean &amp; fast
- ✔ No dependencies required

## 🔧 How to Use
1. Open **Task Manager**  
2. Go to the **Details** tab  
3. Copy the **exact process name** (example: `RobloxPlayerBeta.exe`)  
4. Run the dumper  
5. Enter the process name when asked
   
   ## 📂 Output
The tool will generate:
- `modules.txt` – list of modules  
- `offsets.txt` – dumped offsets  
- `log.txt` – full debug information
